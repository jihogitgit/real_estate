### `DdayBanner` 동작 방식

```
props: priority1Date, applyEnd, winnerDate, contractStart, moveInMonth

- useEffect로 클라이언트에서 오늘 날짜 계산 (SSR/CSR hydration mismatch 방지)
- mainRef = priority1Date ?? applyEnd
- D-day <= 0 → "마감" + 회색 배너
- D-day > 0  → "D-N" + 파란 배너
- 우측에 당첨발표/계약/입주예정 표시
```

### `SubscriptionTimeline` 동작 방식

```
5단계: 모집공고 → 특별공급 → 1순위 접수 → 당첨자 발표 → 계약

- null 날짜인 단계는 생략
- done (과거) → 회색 점
- today (오늘) → 빨간 점 + "(진행중)"
- future (미래) → 파란 점
- useEffect로 today 캡처 (props 변경 시 재계산 가능)
```

### `ApartmentChecklist` 동작 방식

```
공통 항목 4개 + 조건부 항목 1개

priceCap === true  → "분양가상한제 적용 — 실거주의무·전매제한 최장 10년 확인"
priceCap !== true  → "전매제한 기간 공고문에서 확인"

하단 면책 문구: "본 체크리스트는 참고용입니다..."
```

### `IcsDownloadButton` 동작 방식

```
날짜가 하나도 없으면 null 렌더링 (버튼 숨김)

클릭 시:
1. iCalendar 형식 문자열 생성 (RFC 5545 준수)
   - DTEND = DTSTART + 1일 (all-day 이벤트 규격)
   - UID 포함 (CalDAV 호환)
2. Blob URL 생성 → <a> 클릭으로 다운로드
3. setTimeout 100ms 후 URL 해제 (Safari 호환)

이벤트: 1순위 접수 / 당첨자 발표 / 계약 시작
```
